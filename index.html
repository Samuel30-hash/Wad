Course Description

This course introduces students to web technologies- HTML, CSS, Javascr frameworks; javascript frameworks. Students will learnt to implement modern web pages using HTML and CSS framework. the course also introduces students to full stack Web application programming usin javascript

Course Objectives

  1. Desbcribe web technologies (HTML, CSS$ javascript),CSS frameworks, and javascript frameworks.
  2. Implement a simple but a modern web page wih HTML and CSS
  3. Build a simple but functional web application using javascript

Learning outcomes

  • Use web technologies(HTML, CSS $ javascript frameworks.
  • Design and create a simple but functional frameworks.
  • compose and build a simple functional web framework.
